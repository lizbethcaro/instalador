

	IMPORTANTE


1. Para correr de forma exitosa el proyecto debe cumplir con
   los siguientes requisitos

   Servidor de datos: Mysql 5.7+ - MariaDB 10.4+
   Apache 2.4.35 VC15
   PHP 7.2+

2. Debe importar el archivo sql de la carpeta DB. sin cambiar cotejamientos

3. Para usar el traductor primero debe asegurarse que las palabras estén
   agregadas a la base de datos (puede consultarlas en el Indice) desde
   la aplicación.

4. en el archivo llamado creacion_view_funcion.sql, están los scripts de la creación de las vistas y las funciones para la BD
